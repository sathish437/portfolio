## ⚡ Core Features

### 🖥️ Desktop OS Experience
* **Window Management System:** Drag-and-drop, minimize/restore, maximize/fullscreen, and close connection controls for modular application windows (About, Projects, Skills, Contact).
* **Interactive Taskbar (Dock):** Real-time clock/date, live weather status, search input to highlight matching desktop icons, active app indicators with pulsing neon dots, and quick-access social links.
* **Magnetic CTA Physics:** A cinematic hero landing page featuring mouse-tracking magnetic button responses.
* **Futuristic Aesthetics:** Moody luxury design utilizing glassmorphism, depth shaders, atmospheric lighting, neon border glows, and custom cursors.

### 📱 Mobile Pager Experience
* **Horizontal Swipe Pages:** Native horizontal touch/swipe navigation using CSS snap scroll points.
* **Active Node Indicators:** Custom dot navigation at the top of the viewport reflecting active sections in real-time.
* **Quick Sheets:** Bottom sheet components for focused section viewing.

## 💻 Local Setup & Development

Follow these steps to run the project on your local machine:

### Prerequisites
* [Node.js](https://nodejs.org/) installed (v18+ recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sathish437/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Launch development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
